Importer des destinations

Créez ou mettez à jour plusieurs destinations en une seule fois depuis un fichier Excel

Comment ça marche
  • 1
    Exportez d'abord les destinations existantes (bouton "Exporter Excel" sur la liste), ou téléchargez un modèle vierge ci-dessous.
  • 2
    Modifiez le fichier dans Excel : par exemple changez la colonne "Gare (code)" pour dupliquer des destinations vers une autre gare, ou ajustez les montants.
  • 3
    Réimportez le fichier ici : chaque ligne (Gare, Ligne, Ville d'arrivée) déjà existante est mise à jour, chaque nouvelle combinaison est créée.
Format attendu
  • Gare (code) doit correspondre à un code de gare existant.
  • Ligne doit correspondre au nom exact d'une ligne active déjà créée.
  • Active accepte "Oui" / "Non".
  • Les lignes en erreur (gare/ligne introuvable, montant invalide) sont ignorées et listées après l'import ; les autres sont importées normalement.
